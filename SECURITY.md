# Politique de Sécurité / Security Policy

## Signaler une Vulnérabilité / Reporting a Vulnerability

Si vous découvrez une vulnérabilité de sécurité, merci de la signaler par email à l'adresse suivante / If you discover a security vulnerability, please report it via email to:

[contact@ygdrasil.com](mailto:contact@ygdrasil.com)

Ne créez pas d'issue publique sur GitHub. / Please do not create a public GitHub issue.

### Attentes / Expectations

- Vous recevrez un accusé de réception sous 48 heures / You will receive an acknowledgment within 48 hours
- Nous travaillons à reproduire et valider le rapport / We will work to reproduce and validate the report
- Un correctif sera préparé et publié dès que possible / A fix will be prepared and released as soon as possible
- Nous vous tiendrons informé de l'avancement / We will keep you informed of progress

### Processus / Process

1. Rapport envoyé / Report submitted
2. Confirmation de réception / Acknowledgment (48h max)
3. Analyse et reproduction / Analysis & reproduction
4. Correctif préparé / Fix prepared
5. Publication / Release
6. Annonce publique / Public disclosure (après correctif / after fix)

## Versions Supportées / Supported Versions

| Version | Supportée / Supported |
|---------|----------------------|
| >= 1.0.0 | ✅ |
| < 1.0.0 | ❌ |
