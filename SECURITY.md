# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via email to:

**contact@ygdrasil.com**

Please do not create a public GitHub issue.

### Expectations

- You will receive an acknowledgment within 48 hours
- We will work to reproduce and validate the report
- A fix will be prepared and released as soon as possible
- We will keep you informed of progress

### Process

1. Report submitted
2. Acknowledgment (48h max)
3. Analysis & reproduction
4. Fix prepared
5. Release
6. Public disclosure (after fix)

## Supported Versions

| Version | Supported |
|---------|-----------|
| >= 1.0.0 | ✅ |
| < 1.0.0 | ❌ |
